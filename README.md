# Magister

Веб-платформа для дистанционного изучения информатики русскоязычными и иностранными студентами подготовительного факультета.

Курс основан на печатном пособии:

> Е. В. Степаненко, И. Т. Степаненко, Е. А. Нивина. Рабочая тетрадь для иностранных студентов подготовительного факультета. Тамбов: Издательство ФГБОУ ВО «ТГТУ», 2018.

Проект адаптирует материалы пособия в формат интерактивного сайта: словари с озвучкой, упражнения для самопроверки, интерактивные задания и тренажёры по Pascal.

## Возможности

- 24 основных занятия по информатике.
- Дополнительные занятия, включая AutoCAD и тест по Word/Excel.
- Озвучка терминов средствами браузера.
- Интерактивные задания: да/нет, выбор ответа, ввод текста, сопоставление пар, группировка слов, drag and drop.
- Pascal-тренажёр для занятий 19-24: подсветка синтаксиса, запуск программ, задания на прогноз вывода, пропуски и порядок строк.
- Сохранение прогресса в `localStorage` без регистрации и серверной базы данных.

## Технологии

- PHP без фреймворков и базы данных.
- Bootstrap 5, подключён локально.
- Vanilla JavaScript.
- Контент уроков хранится в PHP-файлах в каталоге `data/`.

## Запуск

Проект рассчитан на обычный PHP-хостинг. Для локального запуска достаточно открыть каталог через встроенный PHP-сервер:

```bash
php -S localhost:8000
```

На этой рабочей машине уже есть portable PHP:

```text
D:\MAGISTER\MagisterTest\vendor\php\php.exe
```

Если обычная команда `php` не найдена в `PATH`, запускайте сайт так:

```powershell
D:\MAGISTER\MagisterTest\vendor\php\php.exe -S 127.0.0.1:8000 -t D:\MAGISTER\Magister
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:8000
```

Важно: в проекте используются абсолютные ссылки вида `/assets/...`, поэтому сайт должен открываться из корня домена или локального сервера.

## Проверки

В проекте есть проверки без внешних npm-зависимостей:

```bash
npm test
```

Команда запускает регресс-тесты учебного Pascal-интерпретатора.

Для синтаксической проверки PHP-файлов нужен установленный PHP CLI:

```bash
npm run lint:php
```

Скрипт `lint:php` сначала ищет portable PHP по пути `D:\MAGISTER\MagisterTest\vendor\php\php.exe`, а затем пробует обычную команду `php`.

## Ограничения

- Прогресс хранится только в браузере пользователя.
- Ответы заданий для самопроверки находятся на клиенте в HTML/JS. Для зачётного тестирования нужна серверная проверка.
- Pascal-тренажёр поддерживает учебное подмножество языка: переменные, константы, условия, циклы, массивы, базовые операции и встроенные функции. Пользовательские процедуры/функции, записи, файловый ввод-вывод и полноценный `readln` не реализованы.
